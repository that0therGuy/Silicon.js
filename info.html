General:

1. Ever felt that online code editors always delay your work by constantly showing ads, popups or asking to login or signup.
If that is the case, this code editor is for you!
2. Run the code by pressing the top heading of the Editor.
3. Code is saved after it is run.
4. Lastly, the website allows you to take full advantage of the eval() function, you can make changes to the website to your will.
You have complete independence over the website's look and feel.

Editor:

5. Click inside the editor and press F11 to make editor go fullscreen.
6. Lodash.js, Math.js, Danfo.js (as dfd), Sci.js, Compromise.js (as nlp) and Tensorflow.js (as tf) support available baked right in the editor.
7. 'Scroll to' asks for a numerical input for scrolling to a specific line. If the operation is successful, it will blink green otherwise it will remain the same.
It executes automatically.

File management:

8. Use Blob and FileReader to create files, add content and use it for analysis.
9. Use helper.fileRegister(name,type [for example: text/csv]) and helper.fileDelete(name) to manage files (from blob) to view them at files section.
Once registered, the register code can be removed.
10. Use helper.getFile(name) to get content from a registered file and helper.deleteAll() to delete all registered files.
11. You can change file content directly from file section once registered.

5. Beta Features (if enabled):

1. Toggling false or true by clicking them in the editor
2. Swiping on number values to change them based on the swipe direction.
