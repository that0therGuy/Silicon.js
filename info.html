V1.6 Info:


General:

1. Ever felt that online code editors always delay your process by constantly showing ads, popups or asking to login or signup.
If that is the case, this code editor is for you!
2. Run the code by pressing the top heading of the Editor.
3. Code is saved after it is run.
4. Lastly, the website allows you to take full advantage of the eval() function, you can make changes to the website to your will.
You have complete independence over the website's look and feel.

Editor:

5. Click inside the editor and press F11 to make editor go fullscreen.
6. Lodash.js, Math.js, Danfo.js (as dfd), Compromise.js (as nlp) and Tensorflow.js (as tf) support available baked right in the editor.

File management:

7. Use Blob and FileReader to create files, add content and use it for analysis.
8. Use helper.fileRegister(name,type [for example: text/csv]) and helper.fileDelete(name) to manage files (from blob) to view them at files section.
9. Use helper.getFile(name) to get content from a registered file.
