V1.5 Info:

1. Ever felt that online code editors always delay your process by constantly showing ads, popups or asking to login or signup.
If that is the case, this code editor is for you!
2. Run the code by pressing the top heading of the Editor.
3. Code is saved after it is run.
4. Click inside the editor and press F11 to make editor go fullscreen.
5. Lodash.js and Math.js support available baked right in the editor.
Beta Features (if enabled):

1. Toggling false or true by clicking them in the editor
2. Swiping on number values to change them based on the swipe direction.
